# Artemis 2 | Simuladores AI (Missão Lunar) 🚀

Projeto especial criado em **1 de Abril de 2026** para simular a jornada orbital da missão **Artemis 2**. O portal central permite aos usuários escolherem entre três experiências distintas, cada uma orquestrada por um motor de inteligência artificial de última geração.

## 🌌 Visão Geral

Este projeto é uma demonstração de interfaces premium para exploração espacial, utilizando tecnologias modernas de front-end para criar uma experiência imersiva:

- **Glassmorphism**: Design sofisticado com transparências e desfoque de fundo.
- **Interatividade**: Fundo dinâmico que reage ao movimento do mouse.
- **Animações Fluidas**: Micro-animações e transições suaves para uma sensação de "software de missão real".
- **Design Adaptativo**: Funciona perfeitamente em telas desktop e dispositivos móveis.

## 🤖 Simulações Disponíveis

1.  **GPT 5.4 PRO**: Foco em precisão orbital bruta, cálculos de delta-v e telemetria precisa em tempo real.
2.  **GEMINI 3.1 PRO**: Uma interface multimodal com visualização de dados astronômicos em camadas e assistência por voz.
3.  **Claude Opus 4.6**: Uma experiência voltada ao bem-estar da tripulação, logística de sobrevivência e ética em território inabitável.

## 🛠 Tecnologias Utilizadas

- **HTML5 & CSS3**: Estrutura e estilização pura para máxima performance.
- **JavaScript (Vanilla)**: Lógica de animação de partículas e interação galáctica.
- **Google Fonts (Outfit)**: Tipografia geométrica e futurista.
- **Artemis Core Assets**: Imagens e logos oficiais (simulados).

## 🚀 Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno. Recomendamos o uso de monitores ultra-wide para uma experiência de cabine de astronauta completa.
